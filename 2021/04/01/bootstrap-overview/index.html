Bootstrap

https://www.bootstrapcdn.com/ 에 들어가 CDN 주소를 복사해 작업하는 파일에 적용한다. 따로 스타일을 작성하지 않아도 부트스트랩이 제공하는 클래스를 적용하면 그대로 반영이 된다. 다운받아 사용 가능.

3와 4의 차이

  • CSS와 화면 그리드에서 px에서 em으로 단위 변경
  • IE8 이하 버전 지원 종료

3을 쓰는 이유

  • 관리자 페이지 템플릿
  • 위지윅 에디터
