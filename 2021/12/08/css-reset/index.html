CSS의 절대 권력, 초기화

2021-12-08
HTML & CSS / CSS
  • 대부분의 초기화 스타일은 쓸모 없거나 덮어쓴다.
  • 개발자 도구에서 coverage를 검색해 사용되지 않는 CSS를 확인할 수 있다.

넣으면 적절할 것 같은 코드들

  • 클래스로 reset 하는 경우가 좋을 수도 있다.
    • 클래스:where(선택자): 선택자 정수, 선택자의 특이성을 증가시키지 않음
    • ex) class:where(ol, ul) {list-style: none;} -> ol, ul에 클래스가 들어가있는 곳만 초기화
    • is도 같은 기능을 하고 있지만, 특이성을 증가시킴
    • is, where은 아직 삼성 브라우저에서 작동하지 않음.
