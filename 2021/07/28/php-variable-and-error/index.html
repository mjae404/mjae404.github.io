에러발생

보통 php 파일은 에러가 나면 어디서 에러가 났는지 알려주지 않는다. 그렇기 때문에 파일에서 확인하고 싶은 경우 따로 설정을 해주어야 한다. (물론 visual studio에서도 따로 에러 확인을 할 수는 있다.)

  1. C:\Bitnami\wampstack-8.0.8-0\apache2\php\php.ini을 연다.
  2. display_errors를 검색한다.
  3. display_errors = on으로 설정을 변경한다.
  4. Bitnami WAMP Stack에서 웹서버를 리부팅한다.
  5. 다시 localhost를 확인하면 에러가 어디서 발생했는지 확인할 수 있다.

다 작업을 했을 경우 다시 설정을 off로 해두는 것이 좋다.
