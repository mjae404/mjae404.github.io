н”„лЎңм қнҠё кө¬мЎ°

  • Project Root : мөңмғҒмң„ н”„лЎңм қнҠёлӘ… л””л үнҶ лҰ¬лҠ” Project Rootлқјкі  н•ңлӢӨ.
  • manage.py : Django м•ұмқ„ л§Ңл“Өкұ°лӮҳ, лҚ°мқҙн„°лІ мқҙмҠӨ кҙҖл Ё кё°лҠҘмқ„ мҲҳн–үн•ҳкұ°лӮҳ, к°ңл°ң м„ңлІ„лҘј мӢӨн–үн•ҳлҠ” л“ұмқҳ м—¬лҹ¬кё°лҠҘмқ„ мӢӨн–ү н•  мҲҳ мһҲлҠ” н”„лЎңм қнҠё кҙҖлҰ¬лҘј мң„н•ң лӘ…л № мң нӢёлҰ¬нӢ°. django-adminмқҙ лҸҷмқјн•ң кё°лҠҘмқ„ н•ңлӢӨ.
  • ProjectApp (Django Root) : Django н”„лЎңм қнҠёлҘј мң„н•ң м—¬лҹ¬ нҢҢмқјмқҙ лӢҙкІЁ мһҲлҠ” л””л үнҶ лҰ¬.
