mySQL Primary Key (기본키)

2022-02-04
Data Base / SQL / mySQL

Primary Key: 테이블에서 특정 row 하나를 식별하는 역할.

Natural Key

실제로 어떤 개체가 갖고 있는 속성을 나타내는 컬럼이 Primary Key가 되었을 때를 의미한다.

Surrogate Key

  • 어떤 개체의 실제 속성은 아니지만 Primary Key로 쓰기 위해 추가한 컬럼
  • 보통 1부터 시작해서 1씩 증가하는 정수값을 갖는다.
  • Auto Increment 속성을 컬럼에 설정하면, 해당 컬럼에 관해서는 DBMS가 ‘자동으로 증가’하는 값을 넣는다.
