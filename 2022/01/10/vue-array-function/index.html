2022-01-10
JavaScript

1. 배열의 내장 함수 활용

  • map() 함수 메서드: 배열을 대상으로 반복문 (속도 빠름)
  • filter() 함수 메서드: 배열을 대상으로 조건문

실습 과제

map() 함수 메서드를 활용하여 기존 배열 데이터의 결과값을 제곱근으로 배열화시킴

실습 문제

  • tel의 value 값인 번호의 좌측에 ‘010-‘를 추가하여 배열로 받아오기
  • map() 함수 활용
  • 1차 배열의 데이터들은 객체

실습 문제

  • map() 함수를 활용하여 국어와 영어의 total값을 구하라.
