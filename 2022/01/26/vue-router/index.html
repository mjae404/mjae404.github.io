VUE router

2022-01-26
JavaScript / Vue

라우터란 페이지까지 이동할 수 있는 기능을 의미.
라우터를 사용하면 경로와 컴퓨넌트를 등록하면 싱글 페이지 어플리케이션(SPA : Single Page Application)
사용자가 클릭한 화면이 쉽게 이동하도록 도와주는 역할을 담당.
